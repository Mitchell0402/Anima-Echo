# 每日任务与内容设计（20 天 · ~2 小时）

Status: Final
Last updated: 2026-06-28

> 20 个游戏日，每天 ~6 分钟（2 次下矿 + 夜晚操作），总时长 ~2 小时。
> 参数全部使用现有代码数值，无需改动。

---

## 核心参数（沿用现有代码）

| 参数 | 值 | 文件 |
|------|-----|------|
| 每日下矿次数 | **2 次** | `day_night_cycle.gd` |
| 稳定度日衰减 | **-3** | `stability_system.gd` |
| 门票单价 | 50 铜 | `mining_town_scene.gd` |
| 初始钱包 | 50 铜 | `game_runtime.gd` |
| 铁匠赠票 | 2 张（首次对话） | `mining_town_scene.gd` |
| 深层入场券 | 500 铜（累计≥5次解锁购买） | `mining_town_scene.gd` |
| 星辰矿卖出 | -15 稳定度 | `stability_system.gd` |
| 星辰矿赠予 | +15 稳定度 | `stability_system.gd` |
| 普通矿物赠予 | +2 稳定度 | `stability_system.gd` |

### 单日循环耗时（~6 min）

```
白天:
  走向铁匠 → 购票 ×2               ~15 sec
  走向矿洞入口 → 下矿 1              ~2 min
  返回 → 回到小镇
  走向矿洞入口 → 下矿 2              ~2 min
  返回 → 自动进入夜晚

夜晚:
  走向花店 → 鉴定 ×3-5              ~30 sec
  走向商人 → 出售 ×2-3              ~20 sec
  走向 NPC → 对话/送礼              ~30 sec
  走向床 → 睡觉                    ~10 sec

单日总计:                           ~6 min
```

## 每天详细设计

---

### Day 1 · 异乡来客（~6 min）

**叙事阶段**: Stage 0。

| 步骤 | 内容 |
|------|------|
| 1.0 开场旁白 | 14 行黑屏文字 → 自动过渡到小镇 |
| 1.1 引导覆盖层 | 「大家怎么都如此沉默……和他们聊一聊吧。」→ 点击关闭 |
| 1.2 对话 4 NPC | 任务「小镇初识」自动接受，四条 ○ 0/1 进度 |
| 1.3 铁匠赠票 | 与铁匠首次对话结束 → toast：「铁匠青年递给你两张矿洞门票。持有 2 张。」 |
| 1.4 第一次下矿 | 走向矿洞入口 → 普通矿洞 → 浅层矿 |
| 1.5 第二次下矿 | 直接返回 → 继续下矿（门票自动消耗） |
| 1.6 进入夜晚 | 第二次返回时自动触发夜晚 |
| 1.7 夜晚操作 | 花店鉴定（First Reveal 完成）→ 商人出售一部份 |
| 1.8 睡觉 | → Day 2 |

**Day 1 结束时**: 钱包 ~200 铜（50 初始 + 30+20 任务奖励 + 两次采矿 ~100），门票 0，Stage 0。

**NPC 对话**（Stage 0，每位 1 条）:
- 铁匠: 「你也要下去？行吧。我不拦你。反正你们都会下去的。」（first）
- 花店: 「这些花……我记得它们以前是粉红色的。还是说是黄的？」（first）
- 守夜: 「年轻人，这镇子有一种你看不见的重量。」
- 商人: 「做生意就是做生意，别带感情。」

### Day 2 · 循环初习（~5 min）

**叙事阶段**: Stage 0。

| 步骤 | 内容 |
|------|------|
| 2.1 购票 | 铁匠处购票 ×2（-100 铜） |
| 2.2 两次下矿 | 浅层矿 ×2 |
| 2.3 进入夜晚 | 第二次返回触发 |
| 2.4 鉴定 + 出售 | 花店鉴定、商人出售 |
| 2.5 首次送礼 | 走向花店 → 赠予一块铜矿 → 教学弹窗：「将矿物赠予 NPC 可以提升好感度。每人每天限送一次。」 |
| 2.6 公告板 | 查看每日任务，有可交付的顺路交了 |
| 2.7 睡觉 | → Day 3 |

**Day 2 结束时**: 钱包 ~220 铜，Stage 0。首次送礼教学已完成。

**NPC**: Stage 0 daily 第二条。守夜：「矿洞里的声音，你听到了吗？」

### Day 3 · 稳定（~5 min）

**叙事阶段**: Stage 0。

| 步骤 | 内容 |
|------|------|
| 3.1 购票 + 两次下矿 | 浅层矿 ×2 |
| 3.2 夜晚 | 鉴定/出售 → 送礼 → 睡觉 |

**Day 3 结束时**: 钱包 ~260 铜，累计浅层 6 次。稳定性 70 - 3 = **67**（首次被注意到：「那个数字在往下掉。」）

### Day 4-6 · 浅层积累期（各 ~5 min，共 ~15 min）

**叙事阶段**: Stage 0。

三天节奏相同：购票 → 两次浅层矿 → 鉴定/出售/送礼/每日任务 → 睡觉。

| Day | 累计浅层 | 稳定度 | NPC 每日对话策略 |
|-----|---------|--------|-----------------|
| 4 | 8 | 64 | 花店：「最近记性越来越差了。昨天我把洒水壶放进了冰箱。」 |
| 5 | 10 | 61 | 铁匠解锁深层入场券（≥5 次累计）→ 「深层入场券可购买了。」 |
| 6 | 12 | 58 | 守夜：「有时候，最深的恐惧不是黑暗，是你在黑暗中听到的东西。」 |

**Day 5 关键事件**: 累计浅层 ≥5 → 铁匠 popup 中「购买深层入场券（500 铜板）」按钮从灰色变为可用。但 500 铜板对玩家来说还不够——需要再攒 2-3 天。

**Day 4-6 结束时**: 钱包 ~420 铜，浅层 12 次。

**稳定度变化感知**: 70 → 67 → 64 → 61 → 58。玩家开始注意到：**「小镇在变差。虽然很慢，但确实在变差。」**

---

### Day 7-9 · 攒钱期（各 ~5 min，共 ~15 min）

**叙事阶段**: Stage 0。

| Day | 内容 |
|-----|------|
| 7 | 购票 + 浅层矿 ×2 → 鉴/售 → 精炼台首次尝试（精炼 1 块稀有矿，售价翻倍）→ 钱包 ~480 |
| 8 | 浅层矿 ×2 → 自营小店首次尝试（夜晚随机夜客，溢价卖出矿物）→ 钱包 ~550 |
| 9 | 浅层矿 ×2 → **攒够 500 铜！** → 走向铁匠 → 购买深层入场券 → toast：「购买了深层入场券！深层矿洞入口已开放。」 |

**Day 9 结束时**: 钱包 ~100 铜（花掉 500），深层入场券 ×1，累计浅层 18 次。

**NPC 对话**（Stage 0 后期，语气越来越阴郁）:
- Day 7 铁匠: 「和你一样的人来过，又走了。你是第几个？我已经记不清了。」
- Day 8 花店: 「你知道为什么花店没有花吗？其实我也不知道。」
- Day 9 守夜: 「我在等人。一直在这里等着。」

### Day 10 · 深渊初探 ★（~7 min）

**叙事阶段**: Stage 0 → Stage 1。

| 步骤 | 内容 |
|------|------|
| 10.1 购票 | 购票 ×2 |
| 10.2 第一次深入 | 矿洞入口 → **深层矿洞**（需深层入场券）→ 进入 deep_mine.tscn |
| 10.3 深层采矿 | 更暗的环境，微红色调。矿壁更硬、产出更高、有敌人。**挖到 Warm Geode**（脚本首掉保证）。屏幕微震 0.5s → 低沉嗡鸣 → toast：「你感受到手中的石头……是温热的。」 |
| 10.4 第二次下矿 | 浅层矿（补充门票消耗） |
| 10.5 进入夜晚 | 走向花店 → **鉴定 raw_star_geode** |
| 10.6 星辰矿鉴定 | 「这块石头，带着温热的呼吸。」→ star_crystal 入仓库 → **Stage 1！** |
| 10.7 NPC 对话 | 每人触发 Stage 1 first（已在 dialogues.json 中实现） |
| 10.8 睡觉 | → Day 11 |

**Day 10 结束时**: 仓库 star_crystal ×1，**叙事阶段：Stage 1**。

**NPC Stage 1 first 对话**:
- 守夜: 「它感觉到了你。现在你也感觉到它了。」
- 铁匠: 「你也去了。」（漫长沉默）
- 花店: 「好奇怪……我好像闻到了春天。」
- 商人: 「星辰矿？那东西不好出手。但总有人想买。」

### Day 13 · 抉择之日 ★★（~7 min）

**叙事阶段**: Stage 1 → Stage 2。

这是全游戏最重要的转折点。

| 步骤 | 内容 |
|------|------|
| 13.1 | 两次下矿（浅层/深层均可） |
| 13.2 | 进入夜晚 |
| 13.3 | **做出选择** |

**选择 A：卖出星辰矿（恶人线）**

走向商人 → 出售 star_crystal → ~500 铜到手。
- 稳定度: **-15**（~43 → ~28）。稳定度条骤降，颜色从黄变橙。
- 道德: `sold_star_count = 1`，`has_made_first_choice = true` → **Stage 2**

**NPC Stage 2 first_evil**:
- 守夜: 「卖了也好。活得轻松一点，没什么不对。」（语气不轻松）
- 铁匠: 「不多说了。锤子还等着我。」→ **铁匠不再出售门票**
- 花店: 「今天不浇水也没关系了吧。反正花都死了。」→ 放下水壶

**选择 B：赠予星辰矿（善人线）**

走向花店 → 赠予 → 选择 star_crystal。
- **花店眼泪叙事**: 她的眼泪落下来——全镇十年来第一滴眼泪。枯枝闪过绿光。
- 稳定度: **+15**（~43 → ~58）。回升。
- 好感度: +5
- 道德: `gifted_star_count = 1`，`has_made_first_choice = true` → **Stage 2**

**NPC Stage 2 first_good**:
- 守夜: 「你把它还回去了。你……你做了我们不敢做的事。」
- 铁匠: 「我年轻的时候也该有你这样的勇气。也许他们就不会……」

| 13.4 | 与所有 NPC 感受变化 |
| 13.5 | 睡觉 → Day 14 |

### Day 20 · 三种结局 ★★★（~6-8 min）

#### 恶人结局（Stability ≤ 0）

触发于 Day 19-20 结算。覆盖层淡入：

> 「小镇彻底沉默了。」
> 「铁匠铺的锤子声停了。」
> 「花店的水壶落在地上，没有人捡。」
> 「守夜老人的椅子，空了。」
> 「你变得极其富有。但这条街上，再也没有人看你一眼。」

→ 返回主菜单 → 统计面板。

#### 善人结局（Stability ≥ 100 → Boss 战）

触发覆盖层：

> 「地底深处传来一声沉闷的咆哮——不是愤怒，是痛苦。」
> 「铁匠青年走向你：『我等这一刻等了一辈子。带路。』」

→ Boss 战（~3-4 min）：巨兽光团，3 阶段攻击 → 3 轮弱点攻击击败。

善人结局画面：

> 「巨兽碎裂。被吞噬的灵魂像星光一样升起。」
> 「花店少女的枯枝冒出第一抹绿色。铁匠笑了。老人站起来了。」
> 「你只是一个没有理由留下来的矿工。但你留下来了。」

→ 统计面板。

#### 中立结局（20 天未碰星辰矿）

> 「二十天过去了。你始终没有靠近那条通往深处的矿道。」
> 「小镇没有变好，也没有变坏。铁匠照旧打铁，花店照旧浇枯花。」
> 「你只是安静地挖了一段时间的矿，然后离开了。」
> 「这或许也是一种选择。」

→ 统计面板。

总天数 / 总采矿次数 / 星辰矿卖/送数 / NPC 最终好感度 / 稳定度走势 / 善恶标签。

## 稳定度收敛验证

沿用现有参数（DECAY=3, SELL=-15, GIFT=+15, NORMAL=+2）：

| 路线 | 公式 | Day ~18-20 结果 |
|------|------|-----------------|
| **善人** | 70 - 18×3 + 3×15 + ~8×2 = 70 - 54 + 45 + 16 = **77** | 需调整 |
| **恶人** | 70 - 18×3 - 3×15 = 70 - 54 - 45 = **-29** | Day 14 提前触发 |

> [!WARNING]
> 现有参数下，善人线最后稳定度约 77（不够 100），恶人线在 Day 14 就崩了（不到 20 天）。
> 调整方案（二选一，在 Day 10 前决定）：
> - **方案 A**：增厚星辰矿事件。卖/送 ×4-5 次，善人 70-54+75+16=107 ✓，恶人 70-54-60=-44（扩宽稳定度区间？）。
> - **方案 B**：微调衰减。DECAY=2，善人 70-36+45+16=95，恶人 70-36-45=-11。或者降低卖/送幅度。
>
> 此为已知待调优项，不阻塞 Day 1-9 的开发。等到 Day 10 开发深层矿场景时一并敲定。
