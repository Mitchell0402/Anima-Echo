## Day 2：小镇稳定度 + 昼夜循环

**状态**：✅ 已完成 (2026-06-27)

**目标**：稳定度成为全游戏压力源和反馈面，昼夜循环成为节奏控制器。

**变更**：每日下矿次数从 4 次改为 3 次（用户需求）。稳定度和下矿次数在 HUD 实时显示。

### 任务清单

| # | 机制 | 具体内容 |
|---|------|----------|
| 2.1 | **稳定度系统** | 新建 `StabilitySystem` autoload（`scripts/core/stability_system.gd`）。`stability: float 0~100`，初始 70。每日自然衰减 3。卖出星辰矿→-15。赠予星辰矿→+15。赠予普通矿→+2。暴露 `get_enemy_spawn_multiplier()`、`get_detection_range_multiplier()`、`get_noise_threshold_multiplier()`。 |
| 2.2 | **稳定度影响敌人** | `scripts/enemies/enemy_ai.gd` 读取 StabilitySystem 系数：`_effective_exposure_range()`、`_effective_detection_range()`、`_effective_suspect_range()`、`_effective_noise_threshold()` 替换硬编码范围值。 |
| 2.3 | **昼夜循环** | 新建 `DayNightCycle` autoload（`scripts/core/day_night_cycle.gd`）。每次「矿洞→城镇」返回=进入夜晚阶段。夜晚结束后=新的一天（稳定度衰减触发）。夜晚阶段内矿洞入口关闭，只能访问 NPC。矿工 NPC 夜晚弹出「结束夜晚」按钮。 |
| 2.4 | **每日下矿次数限制** | 浅层每日免费 **3 次**（从 4 改为 3）。每次下矿消耗 1 次配额。夜晚结束后重置。累计次数记录用于深层门控。 |
| 2.5 | **小镇色调变化** | 根据稳定度动态调整城镇场景 `modulate`（低稳定→暗沉偏冷，高稳定→暖亮）。 |
| 2.6 | **HUD UI 显示** | 城镇 HUD 新增稳定度彩色条（红→黄→绿）和「第 N 天 | 剩余下矿 N 次/夜晚」文字。

## Day 3：三大叙事 NPC + 好感度 + 阶段对话

**状态**：✅ 已完成 (2026-06-27)

**目标**：功能性 NPC 升级为叙事角色，建立对话推进系统和好感度体系。

**变更**：新增 4 个叙事 NPC（铁匠青年、花店少女、商人、守夜老人），任务公告板独立为物体。

### 任务清单

| # | 机制 | 具体内容 |
|---|------|----------|
| 3.1 | **叙事 NPC 替换** | `town_npc_interactor.gd` NPC ID 从 miner/buyer/identifier/task_clerk 改为 blacksmith/florist/buyer/elder。铁匠青年=矿洞入口+入场券+装备；花店少女=鉴定+送礼；商人=矿石出售；守夜老人=剧情。公告板独立为可交互物体（`TASK_BOARD_POSITION`）。 |
| 3.2 | **好感度系统** | 新建 `scripts/narrative/npc_affection.gd`（RefCounted，由 GameRuntime 持有）。`affection: 0~100`，送礼+1/+3/+5（普通/稀有/星辰）。每日每人限 1 次，夜晚重置。 |
| 3.3 | **叙事阶段对话表** | 新建 `data/narrative/dialogues.json`。4 个 NPC 各 4 个阶段（0-3），每阶段含 first/daily/first_evil/first_good/evil/good 对话行。 |
| 3.4 | **对话 UI** | 新建 `scripts/narrative/dialogue_ui.gd`。底部面板，左侧 NPC 立绘，右侧逐字打字效果。按 E 前进，最后一条关闭。 |
| 3.5 | **深层入场券门控** | 铁匠 NPC 在累计浅层挖矿 < 5 次时灰显深层入场券按钮，≥5 次解锁购买（500 铜板）。深层矿场景为 Day 5 任务。 |

## Day 4：善恶双线深度

**状态**：✅ 已完成 (2026-06-27)

**目标**：专属装备区分路线体验，经济系统补完深度。

**变更**：装备系统用文字标签占位精灵（无图片资源）。夜店筛选从 catalog 内嵌逻辑替代 get_night_customers。

### 任务清单

| # | 机制 | 具体内容 |
|---|------|----------|
| 4.1 | **善恶专属装备** | `data/equipment/equipment.json`：恶人线 3 件（贪欲镐+血肉护符+负重囊），善人线 3 件（静心石+回溯碎片+共情光环）。`scripts/player/equipment_system.gd`：RefCounted 装备管理（4 槽位、购买、赠予、效果查询）。 |
| 4.2 | **基础装备升级** | 降噪镐 L1/L2/L3（noise_reduction 15/30/50%）、扩容背包 L1/L2/L3（+4/8/12 热栏格）、精准护符 L1/L2/L3（QTE 成功区 +8/16/25°）。铁匠「装备商店」按钮。 |
| 4.3 | **每日任务刷新** | `catalog.json` 新增 6 条每日任务模板（daily_pool）。`task_service.refresh_daily_tasks()` 每次夜晚结束随机抽 3 条启用。`game_catalog.get_tasks_for_pool("daily_pool")` + `get_night_customers()`。 |
| 4.4 | **矿石精炼系统** | `scripts/town/refine_workstation.gd`：独立物体标（坐标 900,500）。消耗铜板（30/80/200）精炼矿物→售价×2、送礼效果×2。 |
| 4.5 | **自营小店** | 商人 NPC 夜晚弹出「开店」按钮。随机刷 1 位夜客（Lux/Ven/Iria），匹配偏好标签溢价卖出（perfect 1.40-1.55x）。 |
| 4.6 | **装备效果接入** | `enemy_ai.gd`：伤害减免。`move_controller.gd`：固有降噪+行走降噪。花店少女赠予星辰矿 ≥2 且好感度 ≥50 自动回赠善人装备。 |

## Day 5：结局 + Boss 战

**目标**：三条终点线，让游戏有始有终。

### 任务清单

| # | 机制 | 具体内容 |
|---|------|----------|
| 5.1 | **恶人结局** | 稳定度降至 0→触发恶人结局动画：小镇彻底崩坏，NPC 变为不可交互的空壳，色调全黑。播放尾声文字，返回主菜单。 |
| 5.2 | **善人结局 — Boss 战前置** | 稳定度升至 100→触发「巨兽苏醒」事件。矿洞底层开启最终区域。 |
| 5.3 | **Boss 战** | 巨兽（以意识为食的能量体）作为最终 Boss。战斗机制：阶段性攻击（意识波、矿脉突刺、召唤小型魔物）。弱点暴露阶段玩家需要靠近攻击。击败后触发善人结局动画。 |
| 5.4 | **中立结局** | 如果玩家只挖浅层 30 天以上且从未碰过星辰矿→触发中立结局：小镇维持现状，NPC 送别玩家（他们仍然麻木，但感谢你没有伤害他们）。 |
| 5.5 | **结局统计面板** | 通关后显示统计：共挖了多少矿、卖出/赠予了多少星辰矿、每位 NPC 的最终好感度、小镇稳定度走向图。 |
